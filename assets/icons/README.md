# Sophia Chat Icons

Add the following icon files to this folder:

- `sophia-default.png` - Default Sophia icon
- `sophia-purple.png` - Purple variant
- `sophia-blue.png` - Blue variant
- `sophia-green.png` - Green variant

**Recommended specifications:**
- Size: 200x200 pixels
- Format: PNG with transparent background
- Shape: Circular or will be displayed in a circle
