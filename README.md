# Sophia Chat - WordPress Plugin

Add the Sophia Chat bubble to your WordPress website. Sophia provides real-time, accessible, anonymous support and resources for individuals affected by domestic violence.

## Installation

### Option 1: Download from GitHub

1. Download the latest release from [GitHub](https://github.com/SpringACT/sophia-plugin/releases)
2. In your WordPress admin, go to **Plugins > Add New > Upload Plugin**
3. Upload the downloaded `.zip` file
4. Click **Install Now**, then **Activate**

### Option 2: Manual Installation

1. Download or clone this repository
2. Upload the `sophia-plugin` folder to `/wp-content/plugins/`
3. In WordPress admin, go to **Plugins** and activate **Sophia Chat**

## Configuration

1. Go to **Settings > Sophia Chat**
2. Enter your **Integration ID** (contact SpringACT if you don't have one)
3. Choose your preferred **icon style**
4. Set **page visibility** options if needed
5. Click **Save Changes**

## Settings

| Setting | Description |
|---------|-------------|
| **Integration ID** | Your unique Sophia Chat identifier (required) |
| **Region** | Select US or EU based on your users' location |
| **Chat Icon** | Choose from preset icons or use a custom image |
| **Show Chat On** | Control which pages display the chat bubble |
